# 📝 Todo App — React + Vite + Tailwind CSS

A fully functional Todo Application built with React, Vite, and Tailwind CSS during my MERN Stack Internship at SVVV, Indore.

## ✨ Features
- Add new tasks
- Mark tasks as complete / incomplete
- Delete tasks (hover to reveal)
- Filter — All / Active / Completed
- Clear all completed at once
- Remaining task counter
- Responsive UI

## 🛠️ Tech Stack
- React.js
- Vite
- Tailwind CSS
- JavaScript ES6+

## 🧠 React Concepts Used
- useState — todos array and filter state
- Props — passing data and functions to child components
- .map() — rendering todo list
- .filter() — filtering todos by status
- Conditional rendering — empty state message
- Spread operator — adding new todo
